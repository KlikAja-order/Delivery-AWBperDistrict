📝 AWB per District + Tabel Tambahan

Cara penggunaan:
1. Upload file hasil download (format pipe | )
2. File berasal dari menu Awere → Receiver Information
3. Data yang memenuhi kriteria akan dipindahkan ke tabel tambahan per nama (Arif, Rudi, Azi, Gigih, Fitli, Ahmad)
4. Klik HD Download pada setiap tabel untuk menyimpan gambar (scale 4)
5. Tabel memiliki lebar kolom tetap (AWB 120px, Nama 160px, Alamat 320px)
6. Tab 1: tabel gelap, Tab 2: tabel putih/cream dengan font gelap
7. Download di tab 2 akan menghasilkan gambar dengan tema terang (sudah diperbaiki)
